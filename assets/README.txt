# assets/

Place your app icons here before building:

| File        | Size      | Used for                        | How to get it                                      |
|-------------|-----------|---------------------------------|----------------------------------------------------|
| icon.png    | 512×512   | General / loading splash        | Copy your existing public/apple-icon.png here      |
| icon.ico    | Multi-size | Windows installer + taskbar    | Convert icon.png at cloudconvert.com → ICO         |
| icon.icns   | Multi-size | Mac installer + dock           | Convert icon.png at cloudconvert.com → ICNS        |

Quick steps:
1. Copy public/apple-icon.png → assets/icon.png
2. Go to https://cloudconvert.com
3. Upload icon.png, convert to .ico → save as assets/icon.ico
4. Upload icon.png, convert to .icns → save as assets/icon.icns
