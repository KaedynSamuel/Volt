## Quick way to generate icons

1. Start with a **1024Ă—1024 PNG** of the Volt logo (the V with gradient)
2. Use https://www.img2icns.com for macOS `.icns`
3. Use https://icoconvert.com for Windows `.ico`
4. Save the 512Ă—512 PNG directly as `icon.png` for Linux
